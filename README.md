# argallery
Картина в AR

Обязательные Фичи:
* Добавление картины на вертикальную плоскость
* Экран выбора рамки
* Загрузка случайной картинки из интернета	

Advanced Фичи:
* Возможность вставить картинку с камеры устройства

Пример API с картинками:
